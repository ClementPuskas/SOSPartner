Commande à connaître par coeur :

php bin/console make:controller                 --  Créer un controller
php bin/console make:form                       --  Créer un formulaire
php bin/console make:entity                     --  Créer une entity

php bin/console make:migration                  -- A effectuer après avoir mis à jour une entité
php bin/console doctrine:migrations:migrate     -- Permet de mettre à jour la base de donnée

php bin/console doctrine:dabatase:create        -- Permet de créer la base de donnée
php bin/console doctrine:schema:create          -- Permet de créer les tables une fois la base créer


Connexion à la BDD :

Changer les valeurs dans le fichier .env ligne 27 sur le format suivant :

DATABASE_URL=mysql://TON_NOM_DUTILISATEUR:TON_MDP@HOTE:PORT/NOM_DE_LA_BDD
